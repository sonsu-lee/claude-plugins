# 敬語ガイド — 職場コミュニケーション向け

## Register Decision Tree

```
文書（仕様書、報告書、正式文書）?
  → Yes → 格式体（だ/である体）
  → No → 上司・社外向けメール?
    → Yes → 丁寧語 + 適切な尊敬語/謙譲語
    → No → 親しい同僚へのDM?
      → Yes → 軽い丁寧語
      → No → 丁寧語 standard
```

## お/ご Prefix Rules

- 和語 → お: お電話、お時間、お名前
- 漢語 → ご: ご連絡、ご確認、ご質問
- 例外: ご飯、お料理、お電話
- 使わない: 技術用語、カタカナ語、自分の行為

## 〜させていただきます — 使い分け

**使う場面:** 相手の許可が必要な行為、謙遜が求められる場面
- 提案させていただきます → OK

**使わない場面:** 単に報告する通常の行為
- 確認しました → ○ (確認させていただきました → ×)
- 送ります → ○ (送らせていただきます → 過剰)

韓国語話者は 드리다 から転移して過剰使用する傾向あり。

## Common Keigo Errors (Korean L1)

| Wrong | Right | Why |
|---|---|---|
| ご連絡してください | ご連絡ください | ご+Vしてください は不自然 |
| 拝見させていただきます | 拝見します | 拝見 は既に謙譲; 二重謙譲 |
| お分かりになりましたか | ご理解いただけましたか | 文語では不自然 |
| ご苦労様です (to 上司) | お疲れ様です | ご苦労様 は目下向け |
