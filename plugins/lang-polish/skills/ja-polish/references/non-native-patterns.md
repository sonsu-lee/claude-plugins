# 韓国語話者に特有の日本語パターン

## 助詞の誤り

| 韓国語 | 誤用 | 正しい日本語 |
|---|---|---|
| 은/는 (topic) | は を全てに使う | が が必要な場面を区別 |
| 이/가 (subject) | が を全てに使う | は が自然な場面を区別 |
| 에서 (location) | で/に の混同 | 状態→に、動作→で |
| 을/를 (object) | を の過剰使用 | 自動詞では不要 |

## は/が の使い分け (韓国語話者向け)

- 新情報の導入: が (「エラーが発生しました」)
- 既知の情報: は (「このエラーは修正済みです」)
- 対比: は (「AはOKですが、Bは問題があります」)
- 存在文: が (「問題がありました」)

韓国語の 은/이 と完全には対応しない。

## 漢語の直訳

韓国語と日本語は漢字語を共有するが、使い方が異なる:

| 韓国語 | 直訳 (不自然) | 自然な日本語 |
|---|---|---|
| 검토 (検討) | 検討してください | 確認してください (確認の方が一般的) |
| 진행 (進行) | 進行します | 進めます |
| 공유 (共有) | 共有します | 共有します / お送りします |
| 확인 (確認) | 確認お願いします | ご確認お願いいたします |

## 文構造の転移

| Korean pattern | Japanese error | Natural Japanese |
|---|---|---|
| 의 の連鎖 | システムの管理の設定の変更 | システム管理設定の変更 |
| 하다 の直訳 | 作業をする | 作業する (を は省略可) |
| 것 같다 の直訳 | 〜のようです (always) | 〜と思います / 〜かもしれません |

## 過剰な敬語

韓国語 존댓말 の影響で日本語の敬語を過剰に使う傾向:

| 過剰 | 適切 |
|---|---|
| ご確認していただけますでしょうか | ご確認いただけますか |
| 〜させていただければと思います | 〜いたします |
| お忙しいところ恐れ入りますが... (Slackで) | 省略可 (メールなら OK) |

## クッション言葉の不足

韓国語では直接的でも問題ない場面で、日本語ではクッション言葉が必要:

| Direct (不自然) | With cushion (自然) |
|---|---|
| 資料を送ってください | お手数ですが、資料をお送りいただけますか |
| 確認してください | ご確認いただけますでしょうか |
| できません | 申し訳ございませんが、対応が難しい状況です |
